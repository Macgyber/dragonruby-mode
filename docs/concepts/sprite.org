** Real usage
:PROPERTIES:
:NAV-TARGET: usage
:END:
In a typical game, sprites are updated on every tick to reflect movement, animation, or state.

** Common errors
:PROPERTIES:
:NAV-TARGET: error
:END:
- Using incorrect paths (check for typos or missing directories).
- Drawing outside the viewport (coordinates must be within 1280x720).
- Forgetting dimensions (width and height are mandatory for correct scaling).
